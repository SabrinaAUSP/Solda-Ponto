# Solda-Ponto
## Descrição
Esse projeto foi criado para a disciplina de Eletrônica para a Computação da turma do Bacharelado em Ciência da Computação (BCC) de 2026, da USP de São Carlos. Este projeto tem como objetivo desenvolver uma solda ponto (solda por corrente) utilizando componentes eletrônicos e materiais adquiridos e integrados pelos membros do nosso grupo. O foco é projetar, montar e testar um equipamento funcional, buscando compreender os princípios elétricos envolvidos, aplicar conhecimentos de eletrônica e buscar uma solução de baixo custo para a realização de soldas por ponto em pequenas aplicações.
